# 🚗 Voyah Launcher - Современный лаунчер для автомобиля

Премиум Android-лаунчер для автомобиля Voyah с современным дизайном и плавными анимациями.

## 🎨 Особенности

- ✨ Современный glassmorphism дизайн sidebar
- 💫 Плавные анимации показа/скрытия панели
- 🎯 Анимированные индикаторы активных кнопок
- 🌙 Премиум темная цветовая палитра
- 📱 Ripple эффекты на кнопках
- 🚀 Entrance анимация при запуске
- 🎨 Эффект свечения и градиенты

## 🚀 Главное меню (НАЧАТЬ_ЗДЕСЬ.bat)

**Запустите файл:**
```
НАЧАТЬ_ЗДЕСЬ.bat
```

Это главное меню со всеми функциями:
- ⚡ Быстрое обновление / отправка
- 📋 Полные меню с опциями
- 📊 Статус проекта
- 🌿 Управление ветками
- 🌐 Быстрый доступ к GitHub

## 📥 Как СКАЧАТЬ изменения с GitHub

### Вариант 1: Быстрое обновление (рекомендуется) ⚡
Просто **дважды кликните**:
```
1_СКАЧАТЬ_ОБНОВЛЕНИЯ.bat
```
Автоматически скачает последнюю версию из `main`.

### Вариант 2: Полное меню обновлений 📋
```
3_ПОЛНОЕ_МЕНЮ_ОБНОВЛЕНИЙ.bat
```
Меню с опциями:
- 📥 Скачать main ветку
- 🔄 Синхронизировать ВСЕ ветки
- 🎯 Скачать конкретную ветку
- 📊 Показать статус
- 🌿 Показать все ветки
- ⚙️ Сбросить изменения (при конфликтах)

## 📤 Как ОТПРАВИТЬ свои изменения на GitHub

### Вариант 1: Быстрая отправка ⚡
```
2_ОТПРАВИТЬ_НА_СЕРВЕР.bat
```
Автоматически отправит все изменения.

### Вариант 2: Полное меню отправки 📋
```
4_ПОЛНОЕ_МЕНЮ_ОТПРАВКИ.bat
```
Меню с опциями:
- 📦 Отправить ВСЕ изменения
- 📋 Выбрать конкретные файлы
- 💬 Добавить описание изменений
- 📊 Показать что изменилось

### Вариант 3: Вручную через командную строку
```bash
git add .
git commit -m "Ваше описание"
git push origin main
```

## 🛠️ Сборка проекта

1. Откройте проект в **Android Studio**
2. Нажмите **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**
3. APK будет в папке `app/build/outputs/apk/debug/`

## 📱 Установка на устройство

### Через Android Studio:
1. Подключите устройство по USB
2. Нажмите **Run** (зеленая стрелка)

### Через ADB:
```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 🐛 Решение проблем

### Проблема: "git pull" не работает
**Решение:** Используйте `update_project.bat` → опция 6 (Сброс изменений)

### Проблема: Конфликт файлов
**Решение:** 
```bash
git stash          # Сохранить ваши изменения
git pull origin main
git stash pop      # Вернуть ваши изменения
```

### Проблема: Gradle build failed
**Решение:** 
1. В Android Studio: **File** → **Invalidate Caches / Restart**
2. Или удалите папку `.gradle` и пересоберите
